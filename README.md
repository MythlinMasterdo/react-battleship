# react-battleship

### Installing Dependencies

From within client folder:

```sh
npm install
```

From within server folder:

```sh
npm install
```

### Setting Up Dev Mode

From within server folder:

```sh
nodemon index.js
```

From within client folder:

```sh
npm start
```
